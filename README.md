## 📌 Sobre o Projeto

Dashboard interativo desenvolvido no Power BI com foco na análise de desempenho acadêmico, matrículas e situação financeira dos alunos de uma universidade. O projeto utiliza uma base com **167 alunos** e **22 variáveis**, permitindo uma visão completa do perfil e desempenho da instituição.

---

## 🎯 Objetivos

- ✅ Criar um dashboard interativo e intuitivo
- ✅ Apresentar os principais KPIs acadêmicos e financeiros

---

## 📊 Dados Utilizados

| Campo | Descrição |
|-------|-----------|
| Aluno / Matrícula | Identificação do aluno |
| Curso / Campus / Turno | Informações de vínculo acadêmico |
| Nota de Aprovação | Desempenho acadêmico |
| Situação | Status do aluno (ativo, evadido, etc.) |
| Mensalidade / Forma de Pgto | Dados financeiros |
| Gênero / Idade / PCD | Perfil demográfico |
| Pegou DP / DPs / Diploma | Indicadores de desempenho |
| Estágio | Situação de estágio |

---

## 📈 KPIs Apresentados

| Indicador | Descrição |
|-----------|-----------|
| Total de Alunos | Quantidade total de matriculados |
| Taxa de Aprovação | % de alunos aprovados |
| Faturaramento | Faturamento por curso |
| Pago ou Não Pago | Situação de pagamento das mensalidades |
| Idade | Idade média por curso |
| Tabela Alunos | Resumo por alunos |
| Alunos | Filtragem Alunos |
| Campus | Filtragem Campus |
| Curso | Filtragem Curso |

## 🛠️ Ferramentas Utilizadas

- **Power BI Desktop** — criação do dashboard
- **Excel** — fonte de dados (167 registros, 22 variáveis)
- **DAX** — criação de medidas e KPIs

---

## 🚀 Como Visualizar

1. Faça o download do arquivo `dashboard-universitario.pbix`
2. Abra com o **Power BI Desktop** (gratuito em [powerbi.microsoft.com](https://powerbi.microsoft.com))
3. Explore o dashboard interativo
